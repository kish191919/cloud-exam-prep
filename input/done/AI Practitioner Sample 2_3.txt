43. 한 헬스케어 플랫폼이 AWS에서 환자 데이터 분석 시스템을 운영하고 있습니다. 여러 국가의 환자 데이터를 처리하므로 각 국가별 법규를 준수해야 하며, 특히 데이터가 어느 국가에 저장되는지와 데이터를 얼마나 보관해야 하는지를 명확히 구분하여 관리해야 합니다. Data Residency와 Data Retention의 차이를 가장 정확하게 설명한 것은 무엇인가요?

1) Data Residency는 데이터가 물리적으로 저장되는 위치와 관련되고, Data Retention은 데이터를 얼마나 오랫동안 보관할지에 대한 정책이다
2) Data Residency는 데이터 암호화 방식(AES, KMS 등)을 정하는 것이고, Data Retention은 IAM 권한을 부여/회수하는 방식이다.
3) Data Residency는 실시간 분석 성능(지연시간)을 최소화하는 네트워크 최적화 개념이고, Data Retention은 데이터 압축률을 높이는 스토리지 최적화 전략이다.
4) Data Residency는 백업 빈도(매일/매주)를 정하는 것이고, Data Retention은 장애 복구 목표(RTO/RPO)를 정하는 것이다.

44. 한 이커머스 회사가 AWS에서 고객 구매 예측 ML 모델을 개발하고 있습니다.데이터 과학팀은 원천 데이터를 그대로 사용하기보다, 구매 날짜에서 요일·월·공휴일 여부를 추출하고, 거래 금액을 로그 변환하며, '최근 30일 구매 횟수' 같은 새 입력값을 만들어 모델 성능을 개선하려 합니다. 머신러닝에서 Feature Engineering을 가장 정확하게 설명한 것은 무엇인가요?

1) Feature Engineering은 원천 데이터에서 모델 입력으로 사용할 특성을 선택·변형·생성하여 모델 성능을 개선하는 과정이다
2) Feature Engineering은 학습된 모델을 프로덕션 환경에 배포하고 API로 제공하는 과정이다
3) Feature Engineering은 학습 데이터 없이 모델이 스스로 정답을 찾도록 하는 비지도 학습 알고리즘이다
4) Feature Engineering은 모델의 하이퍼파라미터(learning rate, epoch 등)를 자동으로 탐색해 최적화하는 과정이다

45. 한 광고 에이전시가 생성형 AI를 활용해 광고용 캐릭터 이미지를 빠르게 제작하고자 합니다. 팀은 학습 데이터로 다양한 캐릭터 이미지를 제공했고, 모델이 얼굴 형태, 표정, 의상 스타일, 색상 조합 같은 시각적 특징의 패턴을 학습하도록 했습니다. 목표는 학습 데이터에 없던 새로운 캐릭터 이미지를 만들어 광고 시안으로 사용하는 것입니다. 이 생성형 모델이 수행할 수 있는 작업으로 가장 적절한 것은 무엇인가요?

1) 광고 이미지에서 로고나 문구 위치를 찾아 좌표로 표시하는 객체 탐지(Object Detection) 를 수행한다.
2) 학습 데이터셋에 없던 새로운 캐릭터 이미지를 생성할 수 있다
3) 여러 장의 캐릭터 이미지를 비교해 동일 캐릭터인지 판별하는 유사도/매칭(이미지 검색·재식별) 을 수행한다.
4) 캐릭터 이미지의 배경과 인물을 픽셀 단위로 구분하는 세그멘테이션(Segmentation) 을 수행한다.
