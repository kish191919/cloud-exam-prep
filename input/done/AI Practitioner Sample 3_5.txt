65. 한 대학교 AI 학과가 학생들에게 강화 학습을 직접 실습할 수 있는 교육 환경을 도입하려고 AWS DeepRacer를 검토하고 있습니다. 교수팀은 도입 전에 다음 사항을 정확히 파악하고자 합니다.
- DeepRacer가 실제 물리 장치인지 여부
- 물리 차량 없이 시뮬레이터만으로도 학습 실습이 가능한지
- DeepRacer의 학습 방식이 어떤 ML 기법에 기반하는지
AWS DeepRacer에 대한 설명으로 가장 올바른 것은 무엇입니까?

1) AWS DeepRacer는 강화 학습(Reinforcement Learning)을 기반으로 하며, Wi-Fi가 지원되는 실제 물리 차량이 제공되고 물리 차량 없이도 시뮬레이터만으로 강화 학습을 실습할 수 있다
2) DeepRacer는 물리 장치가 없는 순수 소프트웨어 서비스이며, 시뮬레이터만으로는 학습이 불가능하고 실제 트랙 장비가 반드시 필요하다. 또한 학습은 지도 학습 기반이다.
3) DeepRacer는 실제 물리 차량이지만, 학습은 사전 수집된 주행 데이터로 학습하는 비지도 학습 방식이며 시뮬레이터는 단순 시각화 용도라서 학습 실습에 사용할 수 없다.
4) DeepRacer는 물리 차량 없이도 가능하긴 하지만, 학습 방식은 강화 학습이 아니라 회귀(Regression) 기반의 전통적 ML 모델을 학습해 조향 각도를 예측하는 방식이다
