1. 한 SaaS 개발팀이 오래된 백엔드 서비스를 유지보수하고 있습니다. 기존 함수들에 대한 테스트 코드와 REST API 설명서를 빠르게 보강하려고 할 때 가장 적합한 AWS 서비스는 무엇입니까?

2. 한 온라인 가구몰이 대량의 상품 소개 문구를 자동으로 작성하려고 합니다. 모든 문구는 브랜드 톤을 유지하면서도 핵심 장점을 자연스럽게 설명해야 합니다. 이 요구사항에 가장 적합한 AI 모델 유형은 무엇입니까?

3. 한 통신사가 고객 해지 예측 모델을 만들었습니다. 학습 데이터에서는 거의 완벽한 성능을 보이지만, 새로운 가입자 데이터에서는 정확도가 크게 떨어집니다. 이 현상을 가장 잘 설명하는 원인은 무엇입니까?

4. 한 결제 플랫폼이 거래 흐름에서 비정상적인 행동 패턴을 스스로 학습해 사기를 식별하려고 합니다. 사람이 일일이 규칙을 만드는 방식보다 더 적합한 접근은 무엇입니까?

5. 한 리테일 기업이 고객을 자동으로 군집화하는 기능, 가격을 자동 조정하는 예측 기능, 그리고 사람이 정의한 규칙으로 승인 여부를 판단하는 기능을 비교하고 있습니다. 이 중 머신러닝이 아닌 접근 방식은 무엇입니까?

6. 한 병원이 흉부 촬영 이미지와 초음파 스캔을 기반으로 이상 징후를 탐지하는 모델을 훈련하려고 합니다. 이런 컴퓨터 비전 모델 학습에 가장 적합한 데이터 유형은 무엇입니까?

7. 한 서비스 운영사가 고객 문의 메일과 채팅 로그를 분석해 감정, 주제, 긴급도를 자동 분류하려고 합니다. 이 자연어 처리 작업에 가장 적합한 AWS 관리형 서비스는 무엇입니까?

8. 한 데이터 분석팀이 새 ML 프로젝트를 시작하며 값의 분포, 결측치, 이상치, 변수 간 관계를 먼저 확인하고 있습니다. 탐색적 데이터 분석의 주된 목적은 무엇입니까?

9. 한 은행이 고객 상환 이력을 바탕으로 신용 위험 예측 모델을 만들고 있습니다. 데이터 수집부터 배포까지 일반적인 ML 파이프라인 단계에 해당하지 않는 것은 무엇입니까?

10. 한 의료 연구기관이 질환 분류 모델을 평가하면서 여러 threshold에서 민감도와 특이도의 균형을 하나의 값으로 보고 싶어 합니다. 가장 적절한 평가지표는 무엇입니까?

11. 한 쇼핑 앱이 과거 주문 이력과 구매 여부 레이블이 포함된 데이터를 이용해 구매 예측 모델을 개발하려고 합니다. 가장 적합한 학습 유형은 무엇입니까?

12. 한 의료 스타트업이 적은 수의 진단 이미지만 보유하고 있어, 일반 이미지로 미리 학습된 모델을 활용하려 합니다. 사전 학습 모델 사용의 가장 큰 장점은 무엇입니까?

13. 한 자동차 부품사가 시각 검사 모델을 만들면서 learning rate, batch size, epoch 수를 일일이 바꾸는 데 많은 시간을 쓰고 있습니다. 최적 하이퍼파라미터를 자동으로 탐색하는 AWS 기능은 무엇입니까?

14. 한 글로벌 기업이 모델 배포 자동화, 규정 준수, 지속적 모니터링을 포함한 운영 체계를 만들려 합니다. MLOps가 다루는 핵심 범위는 무엇입니까?

15. 한 온라인 마켓이 추천 모델의 성공을 평가하려고 합니다. 정확도 외에 비즈니스 관점의 영향을 보려 할 때, 비즈니스 메트릭이 아닌 것은 무엇입니까?
