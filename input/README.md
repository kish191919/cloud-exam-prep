# input/ 폴더 사용법

AWS 시험 문제 텍스트 파일(.txt)을 이 폴더에 저장한 후 `/convert`를 실행하세요.

- 파일 형식: UTF-8 텍스트 (.txt)
- 처리 순서: 파일명 알파벳 순
- 처리 완료 파일: `input/done/` 폴더로 자동 이동
- 한국어·영문 혼재 가능 (파일별 자동 감지)

## 예시

```
input/
  batch1.txt   ← 영문 AIF-C01 문제 50개
  batch2.txt   ← 한국어 문제 30개
  done/
    batch0.txt  ← 이미 처리 완료된 파일
```

## 주의사항

- 이 폴더의 `.txt` 파일은 `/convert` 실행 시 자동으로 처리됩니다.
- 처리된 파일은 `done/` 폴더로 이동되며 삭제되지 않습니다.
- `done/` 폴더의 파일은 재처리되지 않습니다.
